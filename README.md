# Fourth_ModularFiles
Лабораторная работа №4. Вариант 6.<br>Тема: Матрицы.

## №1 - Matrix28
Дана матрица размера **N × M**. Найти минимальный среди максимальных элементов ее столбцов.

## №2 - Matrix55
Дана матрица размера **N × M** (N — четное число). Поменять местами верхнюю и нижнюю половины матрицы.

## №3 - DArray6
Сформировать из некоторой матрицы **A(n,n)** ***нижнетреугольную*** матрицу **B(n,n)** (все элементы выше главной диагонали нулевые).

## №4 - DArray16
Определить, есть ли в данном двумерном массиве размерности **N × M** элемент, равный сумме своих соседей справа, слева, сверху и снизу. Учитывать только те элементы, которые находятся непосредственно справа, слева, сверху и снизу, те, что окружают данный элемент и находятся по диагонали от него не учитывать.
